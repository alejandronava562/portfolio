# Portfolio

A dark, minimal portfolio built with plain HTML, CSS, and JavaScript. Designed to be fully static and GitHub Pages friendly.

## Getting Started

Open `index.html` in your browser to view the site locally.

## GitHub Pages Setup

1. Push this repository to GitHub.
2. In the GitHub repo, go to **Settings â†’ Pages**.
3. Under **Build and deployment**, select **Deploy from a branch**.
4. Choose the `main` branch and the `/root` folder.
5. Click **Save**. Your site will be available at the provided GitHub Pages URL.

## Customize Content

- Update your name, headline, and links inside `index.html`.
- Replace `[Project Name]` and `[Live Demo URL]` placeholders with real project data.
- Swap placeholder images in `/assets` with your own visuals (keep the same filenames or update the references in `index.html`).
